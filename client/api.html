Run a Prediction:

  
  ===
      Run a Stimming prediction based on provided data
  ===

  Definition:
  
  POST /predict
  

  Example Request Body: 

  
                                
post_data = {
    'userId': XXXXXXXXXX,
    'insert': true,
    'accel': [
        {
            'userId': XXXXXXXXXX // user id code
            'x': 1, // floating point value
            'y': 2, // floating point value
            'z': 3, // floating point value
            'timestamp': XXXXXXXXX // (timestamp in ms)
        },
        ...
    ],
    'gyro': [
        {
            'userId': XXXXXXXXXX // user id code
            'x': 1, // floating point value
            'y': 2, // floating point value
            'z': 3, // floating point value
            'timestamp': XXXXXXXXX // (timestamp in ms)
        },
        ...
    ]
}

  
                            
  Example Response: 
                            // Successful DB response.
                            
  {
      inserted: 2,
      prediction: stimming // 'stimming' or other classified activity
  }

Get Stimming Events:

  
  ===
  Get stimming events
  ===

  Definition:
  
  GET /stim/all
  

  Example Response:
  
  {
      userId: XXXXXXX // database-assigned userId.
  }
  

  Example Response:
  // Successful DB response.
  
      Returns a list of stimming entities.
  
  [
      {
        'Name': 'Stim Entity',
        'ID': X,
          ...
      },
      ...
  ]

Get Fitbit Data:

  
  ===
      Get Accelerometer and Gyroscope Data
  ===

  Definition:

  
  GET /accel
  

  Example Request Body: 
  
      {
          userId: 'XXXXXXXXX',// userId of desired user.
          startTime: 'XXXXXXXXX', // start time of accelerometer data events in epoch milliseconds
          endTime: 'XXXXXXXXX'// end time of accelerometer data events in epoch milliseconds
      }
  

  Example Response:
  // Successful DB response.
  
  
 {
      'data': [
          {
            "userId": XXX,
            "x": 1,
            "y": 2,
            "z": 3,
            "timestamp": XXX
          },
        ...
      ]
  }
  

  

 Definition:
  
  GET /gyro
  

  Example Request Body: 
  
      {
          userId: 'XXXXXXXXX',// userId of desired user.
          startTime: 'XXXXXXXXX', // start time of accelerometer data events in epoch milliseconds
          endTime: 'XXXXXXXXX'// end time of accelerometer data events in epoch milliseconds
      }
  

  Example Response:
  // Successful DB response.
  
  
 {
      'data': [
          {
            "userId": XXX,
            "x": 1,
            "y": 2,
            "z": 3,
            "timestamp": XXX
          },
        ...
      ]
  }
  

      Note that the gyro data response takes the exact same form as the accelerometer - aside from differences in units.
  
                        



    

        
Get User Status:

    

    
  
  ===
  Get current Stimming status for a particular userId
  ===

  Definition:
  
  POST /api/schedule
  

  Example Request Body:
  
  {
      userId: XXXX, // userId of the desired user
  }
  

  Example Response:
  
  {
      status: 1, // 1 if detected stimming, 0 if in normal state
  }
